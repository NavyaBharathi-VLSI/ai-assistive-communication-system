# AI-Based Assistive Communication System
AI-based assistive communication system enabling real-time sign language, speech, and text interaction using deep learning and computer vision


##  Overview
This project presents a real-time assistive communication system designed for speech and hearing-impaired individuals.  
It enables bidirectional communication by converting sign language to speech and speech/text to sign using image processing techniques.

---

## Features
- Sign language to text/speech conversion  
- Speech to text conversion  
- Text to speech generation  
- Speech to sign language conversion using image mapping  
- Real-time gesture recognition using camera  
- Bidirectional communication system  

---

## Technologies Used

- Image Processing Techniques  
- Speech Recognition  
- Text-to-Speech (TTS)

## Working
1. Capture hand gestures using camera  
2. Process gestures using image processing techniques  
3. Convert gestures to text/speech  
4. Convert speech/text into corresponding sign images  
5. Display sign language representation using predefined image dataset  
6. Enable real-time bidirectional communication
